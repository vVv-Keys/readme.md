## 🧠 About KeysGuard

**KeysGuard** is a modular, full-stack cybersecurity suite focused on real-time scanning, AI-enhanced recon, and threat intelligence visualization. Designed for red teamers, defenders, and curious minds building hands-on skills.

Built using **Rust, Python, React, TailwindCSS**, and deployable on **Vercel, Docker**, or bare metal.

---

## 🔍 Key Modules

### 🧠 Scanner (Rust)
- Memory/process scanner
- YARA, PE/MZ header detection
- HWID fingerprinting
- Discord webhook + remote sync

### 🌐 Recon Engine (Python)
- Subdomain enumeration
- Port scanning
- SQLi payload injection + metadata scraping
- PDF report generation

### 📊 KeysGuard Dashboard (React)
- Real-time frontend
- ThreatNet visualizer
- Scan logs, alerts, metrics
- Modular and mobile responsive
